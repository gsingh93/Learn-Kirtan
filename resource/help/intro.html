Introduction

Welcome to Learn Kirtan! This software was developed by Gulshan Singh to help people of all ages and skill levels learn kirtan.

This application allows teachers to type shabads in the editor, save them, and distribute them to students, who can then open the shabad file on their computers and watch the notes play on their screen.

If this is your first time using Learn Kirtan, you should read the Getting Started Guide. You can learn about the complete feature set by looking through the Features section.

If you have any questions, comments, or suggestions, please feel free to email me at gulshan@umich.edu.

If you would like to contribute to the Learn Kirtan project, please email me. This project is opensource and any help is welcomed.
