Keyboard Shortcuts

Learn Kirtan includes a number of keyboard shortcuts for quick access to settings and other features. There are two types of shortcuts, menu mnemonics and accelerators.

Mnemonics are keys to press to navigate through the menu quickly. For example, you can see that the "F" in "File" is underlined in the menu. This mean that "F" is a mnemonic for "File". Pressing Alt+F opens the file menu. Once in the file menu, you can see that "O" is underlined in the word "Open". This means that "O" is a mnemonic for "Open". Pressing O (without the Alt) is equivalent to clicking "Open". Note that the Alt key is only used to open the initial menu, but is not needed after this.

The other type of shortcut is an accelerator. An accelerator is a key combination that executes a command. A complete list of accelerators is given below. Note that the accelerator is often given next to the name of the corresponding menu option.

Shortcut Action
Ctrl+A Select all text in the editor
Ctrl+C Copy selected text
Ctrl+X Cut selected text
Ctrl+V Paste the text on the clipboard
Ctrl+Z Undo the last action
Ctrl+Y Redo the last undone action
Ctrl+N Create a new shabad
Ctrl+O Open a shabad
Ctrl+S Save the current shabad
Alt+C Switch to Compose Mode
Alt+E Switch to Edit Mode
F1 Open the Help Menu
