Komal and Theevra Notes

A komal note is a note that is a half step lower than the original note. You can type komal notes by typing an apostrophe (') before the note. A theevra note is a note that is a half step higher than the original note. Theevra notes can be typed by typing an apostrophe after the note.

The only valid komal notes are komal Re ('re), komal Ga ('ga), komal Dha ('dha), and komal Ni ('ni).

The only valid theevra note is theevra Ma (ma').
