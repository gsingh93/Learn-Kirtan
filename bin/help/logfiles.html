Log Files

Every time you run this application a log file is generate. This log file logs every button you click and every shabad you type. This log contains information which will help me figure out the solution to any problems you may experience. If you find any bugs, please email me the log file for the run in which you experienced the bug and describe the bug in detail. Log files can be found in the /log folder. Logs are name as follows:

log_<year>_<month>_<day>_<hour>_<minute>_<second>

Note that this application does not log anything that does not occur in the application. All of your personal information is safe.
